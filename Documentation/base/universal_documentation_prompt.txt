


> **Задача: Создать исчерпывающее руководство по проекту.**
>
> В конце нашей работы над текущим проектом, пожалуйста, выполни следующую задачу:
>
> 1.  **Создай файл:** В корне проекта создай папку `Documentation`, и в ней — файл с названием `project_recreation_guide.md`.
>
> 2.  **Наполни файл** подробным пошаговым руководством, которое позволило бы другому ИИ-ассистенту или человеку с нуля полностью воссоздать текущее состояние проекта.
>
> 3.  **Структура руководства должна включать:**
>     *   **Обзор проекта:** Краткое описание цели проекта, его основных функций и использованных технологий (язык, фреймворки, ключевые библиотеки).
>     *   **Начальная настройка:** Описание исходного состояния или шагов для инициализации проекта (например, "создать стандартный Android-проект" или "выполнить `npm init -y`").
>     *   **Конфигурационные файлы:** Полное содержимое всех ключевых конфигурационных файлов (например, `build.gradle.kts`, `package.json`, `AndroidManifest.xml`).
>     *   **Зависимости:** Четкий список всех необходимых зависимостей и инструкций по их установке.
>     *   **Структура проекта:** Описание финальной структуры папок и файлов.
>     *   **Полный исходный код:** **(Это самый важный пункт)**. Приведи полный и окончательный код для всех ключевых исходных файлов. Не сокращай и не 요약하지 마십시오.
>     *   **Финальная проверка:** Укажи точную команду (или команды) для сборки и проверки проекта (например, `gradlew clean build` или `npm run build`).
>
> Цель этого документа — создать абсолютно полную "резервную копию" контекста проекта, чтобы работу можно было легко проверить или продолжить в любой момент.
>
> Теперь, пожалуйста, создай этот файл для проекта, над которым мы только что работали.
